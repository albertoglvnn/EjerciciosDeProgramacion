# Manejo de Errores y Validación

## Conjunto de programas desarrollados en Python que se centran en:

* Validación de datos ingresados por el usuario
* Control de errores utilizando try y except
* Creación y uso de excepciones personalizadas
* Operaciones básicas y lectura de archivos

## ⚙️ Contenido del proyecto
## 🔢 Calculadora simple

### Incluye operaciones básicas como:

* Suma
* Resta
* Multiplicación
* División

Todas con validación para evitar errores durante la ejecución.

## 🔁 Control de entrada de datos

Se implementa una función como pedir_entero() que asegura que el usuario ingrese valores válidos y evita fallos por datos incorrectos.

## ⚠️ Excepciones personalizadas

### Se definen errores propios para manejar situaciones específicas:

* EdadInvalidaError
* SaldoInsuficienteError
* CalificacionFueraDeRangoError

## 📄 Lectura de archivos

### Se trabajan archivos .txt, incluyendo manejo de errores en caso de:

* Archivo inexistente
* Problemas de acceso
## 🎓 Sistema de calificaciones

### Programa que permite registrar notas con:

* Validación de rango (entre 0 y 100)
* Manejo de errores para entradas inválidas
## 🧠 Conceptos utilizados

En este proyecto se aplican temas clave como:

try, except y finally
Uso de raise para lanzar excepciones
Funciones
Bucles (while)
Validación de datos
## ▶️ Cómo ejecutar

Corre cualquiera de los programas desde la consola con:

python archivo.py
🚀 Propósito

El objetivo principal es reforzar los fundamentos de Python mediante ejercicios prácticos enfocados en situaciones reales, especialmente en el manejo de errores y validación de datos.
